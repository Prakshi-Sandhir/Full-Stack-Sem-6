# 💻 Semester 6 Full Stack Lab

## 📖 Introduction  
This project includes a sequence of 10 hands-on experiments aimed at developing full-stack skills. It progresses from frontend basics in React ⚛️ to backend services using Spring Boot ☕, and finally combines both into a unified application with authentication 🔐 and live data handling ⚡.

## 🧰 Tools & Technologies  
- ⚛️ React.js ecosystem  
- 🔀 Routing with React Router  
- 🧠 State management (Context API, Redux Toolkit)  
- 🎨 UI frameworks: Bootstrap, Material UI  
- ☕ Spring Boot (Java)  
- 🔗 REST API design  
- 🔐 JWT security  
- ⚡ WebSocket communication  

## 📘 Experiment Details  

### 🔸 Experiment 1 – React Essentials  
Understanding components, state, and basic interactivity through simple applications 📝.

### 🔸 Experiment 2 – UI Styling  
Creating adaptive and modern interfaces using popular UI libraries 🎨.

### 🔸 Experiment 3 – Client-side Routing  
Managing multiple views and dynamic URLs within a React app 🔀.

### 🔸 Experiment 4 – Application State  
Handling shared data using modern state management approaches 🧠.

### 🔸 Experiment 5 – Optimization Techniques  
Reducing load time and improving responsiveness with advanced strategies ⚡.

### 🔸 Experiment 6 – React Hooks Usage  
Working with lifecycle and performance hooks effectively 🔁.

### 🔸 Experiment 7 – Context-based Data Flow  
Eliminating deep prop passing with centralized context handling 🔄.

### 🔸 Experiment 8 – Backend APIs  
Developing RESTful services with Spring Boot and performing CRUD operations ☕.

### 🔸 Experiment 9 – Authentication System  
Securing applications with JWT-based login and authorization 🔐.

### 🔸 Experiment 10 – End-to-End Integration  
Connecting frontend and backend, including real-time updates using WebSockets ⚡.
